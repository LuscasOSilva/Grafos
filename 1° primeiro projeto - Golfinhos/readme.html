# Primeiro projeto Golfinhos Para rodar o projeto: Clone o repositorio usando o comando ```bash git clone https://github.com/LuscasOSilva/Grafos.git ``` e em seguida a seguinte linha ```bash type soc-dolphins.mtx | python "Bron-Kerbosch Algorithm Without pivoting.py" ``` Ou ```bash type soc-dolphins.mtx | python "Bron-Kerbosch Algorithm With pivoting.py" ``` Isso retornará os resultados das relações dos Golfinhos no grafo.
