# Primeiro projeto Golfinhos


Para rodar o projeto:
Clone o repositorio usando o comando
```
git clone https://github.com/LuscasOSilva/Grafos.git
```

e em seguida a seguinte linha

```
type soc-dolphins.mtx | python "Bron-Kerbosch Algorithm Without pivoting.py"
```

Ou

```
type soc-dolphins.mtx | python "Bron-Kerbosch Algorithm With pivoting.py"
```

Isso retornará os resultados das relações dos Golfinhos no
grafo.
